1. 기초 TAG 및 설명


장점


  1. 언어 자체가 쉬워 배우기 좋다.
  2. 일상속에 녹아들어 필수가 되었기때문에 중요하다.
  3. Public Domail=저작권이 없다. 누구든 사용 가능하다.

2. 단축키


  1. Ctrl+O = OPEN FILE
  2. Ctrl+S = SAVE
  3. Tab = 들여쓰기
  4. Ctrl+화살표 = 문장 옮기기

3. TAG


  1. STRONG = 진하게 표시
  2. U= 밑줄
  3. H1-6 = 주제문
  4. BR = 줄바꿈 → (추천검색어: HTML new line tag) / NC
  5. P = 단락 → (추천검색어: HTML parahraph tag)
  6. img source = img src = 이미지 넣기
    *img.src="cording.jpg"이미지의 제목이 cording.jpg의 경우
    *img.src="cording.jpg" width="100%"
    → 순서는 상관없으며 이것을 Attribute(속성)이라고 칭한다.
    무료이미지 사용가능 사이트
  7. li = list 목록(목차)만들기
  8. li의 부모 태그! 부모태그는 아이태그 첫과 끝에 꼭 있다. / NC
    ul = unorered list ↔ ol = ordered list
    → 넘버링 할 때 순서의 변화가 생기는 경우 번거로워지므로
    1.2.3. 카운팅을 않고 ol 을 사용하면 자동으로 넘버링
  9. title = 타이틀 설정
  10. meta charset="utf-8" =글자가 깨질 때->본문을 utf-8로 설정하라는 명령
    (utf-8? 유니코드를 위한 가변 길이 문자 인코딩 방식 중 하나라고 함)
  11. body = 본문은 묶기로 사람들끼리의 약속!
  12. head = 본문을 설명하는 것
    *모든 태그는 head와 body 태그로 묶여있어야 함.
  13. html = head/body 보다도 더 높은 태그
  14. !doctype html → 이 문서는 html에 있다 (첫 줄에 관용적으로 사용)/NC
  15. a href="사이트 링크" = 링크
    target="_blank" = 새 탭으로 열고싶을 때
    tilte="설명" = 사이트 열기 전 설명
    *예시) a href="사이트" target="_blank" title="설명" - /a
